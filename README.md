# Mayski Development Project

Первый рабочий вариант одностраничного рекламно-продажного сайта коттеджного посёлка.

## Что входит

- премиальный первый экран;
- блок форматов покупки;
- карточки домов;
- интерактивный генплан;
- кликабельные участки;
- карточка выбранного участка;
- форма заявки через WhatsApp;
- мобильная адаптация.

## Обновление v0.2

Интерактивная карта перестроена не по фото/рендеру, а по PDF-генплану.

- `assets/images/masterplan.png` — новый фон генплана, экспортированный из PDF.
- `js/data.js` — участки с координатами SVG-полигонов.
- Координаты участков получены из векторного слоя PDF: `PDF vector extraction, crop x=500 y=150 w=975 h=1475`.
- В карту добавлены 30 замкнутых участков, которые удалось чисто извлечь из PDF как векторные четырёхугольники.
- `docs/masterplan-vector-overlay-preview.png` — контрольная картинка, где видно, какие полигоны попали в интерактивный слой.

## Важно

Площади участков в `js/data.js` заполнены по видимым подписям на PDF и требуют финальной проверки по исходной ведомости/кадастровым данным.

Цены пока стоят как `null`, поэтому на сайте показывается значение `по запросу`.
Для публикации нужно заменить `price: null` на реальные цены.

Пример:

```js
{
  id: "LOT_001",
  number: "1",
  area: 792,
  price: 4500000,
  status: "available",
  polygon: "239.8,307.0 263.9,76.3 355.8,81.6 364.0,302.5"
}
```
