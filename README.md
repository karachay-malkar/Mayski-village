# Mayski Development Project

Первый рабочий вариант одностраничного рекламно-продажного сайта коттеджного посёлка.

## Что уже есть

- Премиальный первый экран с фото дома.
- Блок преимуществ.
- 3 формата покупки:
  - участок + готовый дом;
  - участок + дом по вашему проекту;
  - только участок.
- Карточки домов/референсов.
- Интерактивный генплан без Mapbox:
  - фоновая картинка генплана;
  - SVG-контуры участков поверх изображения;
  - статусы участков;
  - фильтр по статусам;
  - карточка выбранного участка;
  - подстановка участка в форму заявки.
- Форма заявки через готовое сообщение WhatsApp.
- Адаптивная версия для телефона.

## Как открыть

Откройте `index.html` в браузере.

Для локального сервера можно использовать:

```bash
python3 -m http.server 8080
```

Затем открыть:

```text
http://localhost:8080
```

## Где менять данные участков

Файл:

```text
js/data.js
```

Там находятся:

- список участков;
- цены;
- площади;
- статусы;
- координаты SVG-полигонов;
- контакты;
- карточки домов.

## Что заменить перед публикацией

1. Телефон в `index.html` и `js/data.js`.
2. Email в `index.html` и `js/data.js`.
3. Реальные цены и площади участков в `js/data.js`.
4. Реальные контуры участков на генплане.
5. Реальные изображения домов и общего вида посёлка.
