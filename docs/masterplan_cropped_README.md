# Cropped masterplan bundle

Crop applied from original coordinate system:

- original SVG viewBox: `0 0 3000 2250`
- new SVG viewBox: `0 0 2400 1800`
- crop-box: `left=0, top=0, right=2400, bottom=1800`
- removed right: `600px` = 20%
- removed bottom: `450px` = 20%

The PNG was first resized to `3000×2250`, then cropped to `2400×1800` so it shares the same coordinate system as the SVG.

Use `masterplan_cropped_standalone.html` for phone/local check. Use `masterplan_cropped_site.html` with external PNG/SVG assets for website integration.
